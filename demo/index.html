tw-saved-cards in action

Usage of tw-saved-cards

<tw-saved-cards
  cards="vm.cards"
  selected="vm.selected"
  currency-symbol="vm.currencySymbol"
  amount="vm.amount"
  on-submit="vm.onSubmit()"
  is-processing="vm.isProcessing">
</tw-saved-cards>

where vm.cards is an array of cards specific to the TransferWise API, vm.selected is an object with reference and securityCode of the selected card, vm.currencySymbol and vm.amount are self-explanatory, vm.onSubmit is the function from parent scope to call upon clicking the pay button and vm.isProcessing is a boolean passed from parent scope to change the pay button state to disabled.

Usage of non-TransferWise-API-specific tw-saved-card is possible as well.
